这是一篇用于预览主题效果的样稿。它会覆盖公众号文章里常见的标题、段落、引用、列表、代码块、表格和图片，方便你判断一个主题是否适合当前内容。

## 主题观察重点

切换主题时，可以优先看这几个位置：

- 标题是否醒目，但不会压过正文
- 段落间距是否舒服，长文阅读是否轻松
- 引用块是否适合放结论、提示或补充说明
- 代码块和表格是否清楚，不会显得拥挤
- 图片与正文之间是否有合适的留白

> 一个好主题不需要处处抢眼。它应该让重点自然浮出来，同时保持正文稳定、耐读。

## 正文节奏

微信公众号文章通常会混合说明、观点、步骤和素材。排版主题需要照顾这些不同内容：既能承载技术说明，也能适配经验总结、资讯整理和活动文案。

你可以用 **加粗文字** 强调结论，用 `inline code` 标注命令、变量名或配置项，也可以通过列表把复杂信息拆开。

### 步骤示例

1. 先确认文章结构，包括标题层级、段落顺序和配图位置。
2. 再切换文章主题，观察整体阅读节奏是否合适。
3. 最后选择代码主题。如果选择“跟随主题”，代码块会使用当前文章主题的样式。

### 代码示例

```js
const article = {
  title: '主题预览样稿',
  mode: 'wechat',
  codeTheme: 'noset',
};

console.log(`正在预览：${article.title}`);
```

### 表格示例

| 内容类型 | 观察点 |
| --- | --- |
| 长段落 | 行高、字色、段落间距 |
| 引用块 | 背景、边线、文字可读性 |
| 代码块 | 对比度、换行、复制后的稳定性 |
| 表格 | 边框、单元格留白、信息密度 |
