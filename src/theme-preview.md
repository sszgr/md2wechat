`md2wechat` 是一个纯前端的 Markdown 到微信公众号排版工具。  
它的目标很明确：把编辑、预览、换主题、复制发布整合到同一个页面，减少反复调样式的时间成本。

## 没有找到你喜欢的主题？

如果现有主题还不够贴合你的内容风格，或者你希望定制专属排版方案？

## 这个项目能解决什么问题

很多内容创作者都会遇到这些问题：

- 写完 Markdown 后，还要手动调整公众号样式
- 不同文章风格切换成本高
- 代码块、表格、引用在公众号中的展示不稳定
- 复制粘贴后，最终效果和预览不一致

`md2wechat` 的核心价值是把这些问题前置解决，做到先预览、后发布，尽可能接近所见即所得。

## 核心能力

- 支持常见 Markdown 内容排版
- 提供多种适合公众号的文章风格
- 对标题、引用、列表、代码块有较好的展示效果
- 适合经验总结、教程说明、资讯整理等内容类型
- 能帮助文章在发布前先确认整体阅读体验
- 减少重复调样式带来的时间成本

### 编辑体验

支持边写边看，让内容结构和阅读节奏更容易把控。

### 发布效率

文章风格更统一，能明显缩短从写作到发布的整理时间。

### 可定制能力

同一篇内容可以尝试不同风格，找到更适合自己读者的表达方式。

在线预览：`https://md.sszgr.com`

## 快速使用

1. 在左侧输入 Markdown 内容  
2. 选择更适合文章气质的排版风格  
3. 检查标题、图片、引用和代码块的展示效果  
4. 完成后复制内容并粘贴到公众号编辑器
