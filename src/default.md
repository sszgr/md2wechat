`md2wechat` 是一个纯前端的 Markdown 到微信公众号排版工具。
你可以在这里写作、预览、切换主题、管理图片，并把排版后的内容复制到公众号编辑器。

## 为什么需要它

公众号编辑器对 Markdown、代码块、表格和图片的支持并不稳定。很多文章写完之后，还要反复调整字号、间距、引用块、代码样式和配图位置。

`md2wechat` 把这些步骤提前到预览阶段：先把内容排好，再复制发布，尽量减少来回粘贴和手动修样式的时间。

## 适合哪些内容

- 技术文章：代码、命令、配置和截图比较多
- 经验总结：需要清晰的层级、重点和行动建议
- 周报简讯：信息密度高，适合用列表和表格整理
- 活动文案：需要快速切换不同主题风格

> 好的排版不是装饰，而是让读者更快理解内容。

## 快速上手

1. 在左侧编辑 Markdown。
2. 在右侧预览文章效果。
3. 切换文章主题和代码主题，找到合适的样式。
4. 上传或替换本地图片，确认图片能正常显示。
5. 点击复制，把排版结果粘贴到公众号编辑器。

## 常用写法示例

### 重点说明

可以用 **加粗文字** 提醒读者关注重点，也可以用 `inline code` 展示短命令、变量名或配置项。

### 代码块

```js
function formatArticle(title) {
  return `正在排版：${title}`;
}

console.log(formatArticle('公众号文章'));
```

### 表格

| 场景 | 建议 |
| --- | --- |
| 标题较多 | 使用清晰的二级、三级标题 |
| 代码较多 | 选择合适的代码主题，或让代码样式跟随文章主题 |
| 图片较多 | 先在预览中检查尺寸和位置 |
