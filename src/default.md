`md2wechat` 是一个 Markdown 到微信公众号排版工具。你可以在这里写文章、看预览、换主题、管图片，然后把排好的内容复制到公众号编辑器。

## 为什么用它

Markdown 写作很舒服，但公众号编辑器对代码块、表格、引用和图片的支持并不总是稳定。文章写完之后，常常还要手动调字号、间距、颜色和图片位置。

这个工具把排版提前到发布前：先在浏览器里预览效果，再复制到公众号，减少来回粘贴和重复修样式。

> 排版不只是好看，更重要的是让读者更快抓住重点。

## 三步开始

1. 在左侧编辑 Markdown。
2. 在顶部选择文章主题、代码主题和预览模式。
3. 确认效果后，点击复制并粘贴到公众号编辑器。

如果你希望代码块完全跟随文章主题，可以把代码主题设为“跟随主题”。

## 常见内容示例

### 重点段落

你可以用 **加粗文字** 标出关键结论，也可以用 `inline code` 展示命令、变量名或配置项。

### 列表

- 技术文章适合展示代码、命令和截图
- 周报简讯适合用列表压缩信息密度
- 经验总结适合用标题层级组织结构

### 代码块

```js
function formatArticle(title) {
  return `正在排版：${title}`;
}

console.log(formatArticle('公众号文章'));
```

### 表格

| 场景 | 建议 |
| --- | --- |
| 标题较多 | 使用清晰的二级、三级标题 |
| 代码较多 | 选择代码主题，或让代码样式跟随文章主题 |
| 图片较多 | 发布前检查尺寸、位置和清晰度 |
