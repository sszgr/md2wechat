# md2wechat

`md2wechat` 是一个纯前端的 Markdown -> 微信公众号排版工具。  
它把编辑、预览、换主题、复制发布整合到同一页面，减少反复调样式和来回粘贴的时间成本。

## 这个项目解决了什么问题

很多内容创作者会遇到这些问题：

- 写完 Markdown 后，还要手动调整公众号样式
- 不同文章风格切换成本高
- 代码块、表格、引用在公众号中的展示不稳定
- 复制粘贴后，最终效果和预览不一致

`md2wechat` 的核心价值是把这些问题前置解决，做到先预览、后发布，尽可能接近所见即所得。

## 特性总览

- 纯前端实现，无后端依赖
- 支持常见 Markdown 语法排版
- 提供多种文章风格可快速切换
- 兼顾标题、引用、表格、代码块等常见内容展示
- 适合技术文章、经验总结、资讯整理等公众号场景
- 可在发布前先查看整体阅读效果
- 降低手动调样式和反复粘贴的时间成本

## 快速使用

1. 在左侧输入 Markdown 内容
2. 选择合适的文章风格并检查预览效果
3. 根据内容结构调整标题、段落、引用和配图
4. 完成后复制排版结果并粘贴到公众号编辑器
5. 发布前再检查一次手机端阅读体验
