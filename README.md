# md2wechat

`md2wechat` 是一个纯前端的 Markdown -> 微信公众号排版工具。  
它把编辑、预览、换主题、复制发布整合到同一页面，减少反复调样式和来回粘贴的时间成本。

## 在线使用

`https://md.sszgr.com`

## 这个项目解决了什么问题

很多内容创作者会遇到这些问题：

- 写完 Markdown 后，还要手动调整公众号样式
- 不同文章风格切换成本高
- 代码块、表格、引用在公众号中的展示不稳定
- 复制粘贴后，最终效果和预览不一致

`md2wechat` 的核心价值是把这些问题前置解决，做到先预览、后发布，尽可能接近所见即所得。

## 特性总览

- 纯前端实现，无后端依赖
- Markdown 编辑器（CodeMirror）
- 实时预览（markdown-it）
- 预览主题切换（内置多主题，支持收藏置顶）
- 代码高亮主题切换（highlight.js，多代码主题）
- 主题管理页：查看 / 编辑 / 新增 / 删除 / 另存为
- 主题 CSS 编辑：CodeMirror + CSS 自动补全
- 主题卡片图片预览（可放自定义封面）
- 主题预览文章（`src/theme-preview.md` 可独立编辑）
- 常用主题片段快速插入（标题、引用、代码块、表格等）
- 预览模式切换（手机 / 桌面）
- 一键复制到公众号（复制 HTML + 内联样式）
- 一键全选预览区（手动复制兜底）
- 图片资源管理（文中图片本地引用与替换）
- 本地自动保存（Markdown、主题选择、预览模式、自定义主题）

## 本地开发

```bash
npm install
npm run dev
```

默认打开后即可使用，建议 Node.js `20.19+`（或 `22.12+`）以获得最佳兼容性。

## 构建与预览

```bash
npm run build
npm run preview
```

构建产物在 `dist/`。

## 快速使用

1. 在左侧输入 Markdown 内容
2. 在顶部切换预览主题、代码主题和预览方式
3. 进入“主题管理”按需微调样式
4. 点击“复制到公众号”后粘贴到公众号编辑器
5. 如果浏览器限制复制，可用“全选预览”手动复制兜底

## 主题管理与定制

- 内置主题可直接使用，也可基于它“另存为”自定义主题
- 自定义主题支持新增、编辑、删除
- 建议所有规则以 `.wx-article` 为根选择器，避免污染外部样式
- 主题卡片封面图命名规则：`public/theme-covers/<theme-id>.jpg`
- 主题预览文章可在 `src/theme-preview.md` 中直接改文案与结构

示例 CSS：

```css
.wx-article {
  font-size: 16px;
  line-height: 1.75;
  color: #333;
}

.wx-article h2 {
  border-left: 4px solid #07c160;
  padding-left: 0.6em;
}
```

## 复制策略说明

为尽量适配公众号粘贴场景，复制逻辑会：

- 复制 `text/html` + `text/plain`
- 对预览 DOM 进行内联样式展开
- 去除会限制粘贴后布局的根容器样式（如卡片边框/圆角/宽度限制）
